# 8769bet - AI Agent Gaming Platform

A modern, full-stack gaming platform with AI-powered predictions for Aviator-style crash games.

## Project Structure

```
8769bet/
├── frontend/          # User-facing React application
│   ├── src/
│   │   ├── components/    # Reusable UI components
│   │   ├── pages/          # Page components
│   │   │   ├── home/       # Landing page
│   │   │   ├── auth/       # Login & Register
│   │   │   ├── games/      # Game listing & details
│   │   │   ├── deposit/    # Deposit page
│   │   │   └── profile/    # User profile
│   │   ├── layout/         # Layout components
│   │   └── ui/             # UI primitives
│   └── public/
├── admin/             # Admin dashboard (coming soon)
│   └── src/
└── images/           # Reference images & designs
```

## Frontend Features

### Pages
- **Home** - Landing page with hero, features, games preview
- **Login/Register** - Authentication pages
- **Games** - Game listing with categories and search
- **Game Detail** - Individual game page with demo
- **Deposit** - Payment page
- **Profile** - User dashboard

### Design
- Modern dark theme with neon accents
- Glassmorphism effects
- Smooth animations with Framer Motion
- Fully responsive design
- Custom Tailwind CSS theme

### Tech Stack
- React 19 + Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Heroicons

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Admin Dashboard (Coming Soon)
